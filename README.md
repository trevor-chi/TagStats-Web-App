TagStats Web App
Overview

The TagStats Web App is a custom-built application that serves as the digital endpoint for the TagStats interactive baseball card system. By tapping a TagStats card to a smartphone, users are directed to this web app, where they can explore detailed player information through an engaging and modern interface.

Built using HTML, CSS, and JavaScript, the application transforms traditional baseball statistics into interactive and visually compelling experiences.

Features
Player Profiles

Each player has a dedicated page that includes:

Headshot and biographical information
Career achievements and highlight videos
Detailed tables of raw statistics
Data Visualization

Collection Tracking System

Users can create a profile and track their TagStats card collection
View which cards have been collected and which are still missing
Encourages engagement through completion of the full set

How It Works

A user taps a TagStats card to their NFC-enabled smartphone
The embedded NFC tag opens a unique URL
The web app loads the corresponding player’s profile
The user can explore stats, visualizations, and media content

Purpose

The TagStats Web App is designed to modernize and revitalize the traditional baseball card collecting experience. By integrating physical cards with a digital platform, it introduces a new level of interactivity, engagement, and data accessibility for users.
